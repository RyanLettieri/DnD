How to restore your data

If something goes wrong during the update, you can restore your data by:

  1. Open your D&D character sheet
  2. Open browser developer tools (F12)
  3. Go to Console tab
  4. Paste your backup data as a variable and run:
    5. const backupData = { /* your backup data here */ };
    Object.keys(backupData).forEach(key => {
      localStorage.setItem(key, JSON.stringify(backupData[key]));
    });
    location.reload();

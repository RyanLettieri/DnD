# DnD Character Tracker

DnD Character Tracker is a local-first character management app for Dungeons & Dragons campaigns. It supports multiple characters, character creation, editable sheets, combat tracking, spells, inventory, notes, conditions, and class-specific features.

## Features

- Multi-character dashboard with create, edit, select, and delete actions
- Guided character creation with class, race, background, alignment, ability scores, and story notes
- Character sheet with HP, temporary HP, saving throws, skills, quick actions, conditions, spells, features, notes, and inventory
- Support for the core DnD class list, with deeper class-specific behavior continuing to expand
- Persistent local storage so character data remains available between sessions

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

The app runs locally and stores character data in your browser.

## Development Notes

This project began as a Tortle Artificer tracker and is being expanded into a general DnD character tracker. Some class-specific tabs and automation are still deeper for Artificer and Barbarian than for the rest of the class list.
